# FFmpeg Converter v0.2.0

Prosty konwerter wideo/audio z GUI w Pythonie (Tkinter).
Rozmiar kompresji niektórych plików video z niezauważalną stratą jakości może wynośić nawet 10:1

## Funkcje
- Konwersja do H.265 (CRF 19–33)
- Skalowanie: 4K, Full HD, HD, SD, Mobile
- Ekstrakcja MP3
- Tray icon, paski postępu, debug mode
- Tryby z opisami (bez +)

## Wymagania
- Python 3.8+
- FFmpeg w PATH
- `pip install appdirs pillow pystray`

## Użycie
1. Uruchom `ffmpeg_gui.pyw`
2. Wybierz tryb, CRF, preset
3. Wskaż katalog
4. Kliknij **START**

## Uruchomienie z pliku exe
1. ! install  FFmpeg and PATH
2. https://github.com/Altheny/ffmpeg-converter
3. Click Releases
4. Download ffmpeg_gui.exe
5. Uruchom pobrany plik
6. Wybierz katalog z plikami video
7. Kliknij start
8. Opcje wyprubuj sobie sam ;)
